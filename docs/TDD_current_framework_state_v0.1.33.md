# Current Framework State v0.1.33

v0.1.33 adds admission/control materials after v0.1.32 established data-driven workflow protocols. The framework can now produce certification-eligible overlay plans for the charge-path contract using the new `charge_path_admission_controls_v0_1` suite.

The release still does not certify the charge path-adjustment theorem. It only provides executable materials and controls required to make a certification attempt possible under the contract layer.
