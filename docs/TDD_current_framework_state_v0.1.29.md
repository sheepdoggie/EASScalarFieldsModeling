# Current Framework State v0.1.29

v0.1.29 adds contract-driven overlay synthesis/planning and operator-required-items reporting.

Certification-mode attempts now fail with a machine-readable request for the missing operator-supplied items when the required contract, approved plan, admissible overlays, or signed release materials are absent.

This release does not certify the charge path-adjustment theorem. It improves the pre-run planning architecture needed before any future certification/admission attempt.
