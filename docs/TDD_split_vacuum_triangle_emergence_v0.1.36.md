# TDD: v0.1.36 Split-Vacuum Triangle Emergence Exploratory Runner

Status: exploratory mechanism discovery only. This release does not certify a theorem, charge, lepton supports, or path accommodation.

The runner starts from undefined vacuum and one or more first-association split sites. Each split site produces +1 and -1 branches. Each branch receives three zero-value vacuum associates. SOO processing lifts vacuum-facing associates. Least-gradient association selection is then used to detect emergent sign-coherent triangle motifs. Triangle membership is a readout, not a generator input.

Detected persistent triangles may then be used by a path-discovery stage. Path endpoints are detected formed structures, not preselected theorem endpoints. Center-gradient classification and topology transaction audit remain downstream readouts. Delta L is read only after the transaction audit and is never certification evidence in this exploratory release.

Forbidden generator inputs include predeclared triangle membership, predeclared path endpoints, same/opposite labels, charge labels, handedness labels, target Delta L, expected Delta L, preselected center actions, and preselected path outcomes.
