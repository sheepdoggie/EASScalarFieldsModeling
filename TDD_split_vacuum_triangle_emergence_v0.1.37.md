# TDD: v0.1.37 Split-Vacuum Triangle Emergence Runner Repair

Scope: exploratory mechanism discovery only.

The runner begins from undefined vacuum and first-association split sites. It dynamically splits lifted vacuum points into +/- branch records, records six association slots per split site, proposes associations using pure least-gradient burden over the global generated candidate pool, commits proposals after one cycle, detects triangles from the committed association graph, measures persistence, discovers relational paths through the generated graph, classifies center conditions from generated path scalar profiles, and emits per-control negative-control artifacts.

Forbidden generator inputs include predeclared triangle membership, predeclared path endpoints, same/opposite labels, charge labels, handedness labels, target Delta L, fixed nominal path lengths, same-sign priority, branch-local candidate pools, and endpoint-sign-dispatched center conditions.

Null outcomes such as no triangles, no graph paths, determinate centers, and Delta L = 0 are valid exploratory outcomes.
